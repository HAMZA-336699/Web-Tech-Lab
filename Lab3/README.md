# PHP Admin Dashboard

A simple admin dashboard built with plain PHP, HTML, CSS, and JavaScript.

## Features

- Login with session-based authentication
- Protected dashboard route
- Stats cards and recent orders table
- Logout endpoint
- Responsive layout

## Demo Credentials

- Username: admin
- Password: admin123

## Run in Laragon

1. Put this project in `www/Lab3`.
2. Start Apache in Laragon.
3. Open `http://localhost/Lab3`.

## File Structure

- `index.php` - login page
- `dashboard.php` - protected admin dashboard
- `logout.php` - logout and session destroy
- `config.php` - app data and demo user
- `includes/auth.php` - authentication helpers
- `assets/css/style.css` - styles
- `assets/js/app.js` - simple card animation
